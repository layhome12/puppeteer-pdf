# Puppeteer PDF
HTML to PDF with Puppeteer JS. Minimum requirements is Node JS 18

## Instalasi

Run the command below
```bash
  npm install
```

With Docker Setup
```bash
  docker-compose build
```
```bash
  docker-compose up -d
```

## Running

Run the command below
```bash
  npm run serve
```
